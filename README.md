## 📱 Project Preview

*   **Theme:** Dark elegant mode with premium glassmorphism UI.
*   **Performance:** Ultra-lightweight with zero external frameworks or heavy libraries.
*   **Responsiveness:** Fully optimized for Mobile, Tablet, and Desktop screens.

---

## 🛠️ Technologies Used

*   **HTML5:** Semantic elements for SEO-friendly structure.
*   **CSS3:** Custom properties (variables), backdrop-filters, custom animations, and responsive grids.
*   **Vanilla JavaScript:** Clean DOM manipulation, custom typing effects, form validation, and scroll observers.

---

## 🌟 Key Features

*   **Dynamic Theme Toggle:** Smooth transition between Dark and Light modes.
*   **Scroll Progress Indicator:** Real-time reading progress bar at the top of the viewport.
*   **Custom Interactions:** Ripple click effects on buttons and dynamic intersection animations.
*   **Interactive Counter:** Animated statistics tracking for projects and certifications.
*   **Client-Side Validation:** Bulletproof validation for the built-in contact form.

---

## 📂 Core Sections Included

1.  **Home:** Featuring a professional intro with a live typing animation.
2.  **About Me:** Showcasing academic background, personal information, and key milestones.
3.  **Skills:** Interactive matrix showing technical proficiency levels.
4.  **Projects:** Grid displaying 6 responsive showcase cards with live access points.
5.  **Education:** Beautiful step-by-step vertical timeline layout.
6.  **Certificates & Resume:** Gallery displaying verified academic achievements with preview options.
7.  **Contact:** Functional message form wrapper.
